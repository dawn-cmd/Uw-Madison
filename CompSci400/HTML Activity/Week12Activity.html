Dijkstra's Shortest Path Algorithm

Overview

Dijkstra's Shortest Path Algorithm is a graph search algorithm that solves the single-source shortest path problem for a graph with non-negative edge weights, producing a shortest path tree.

Algorithm Steps

  1. Create a set of unvisited nodes and a set of visited nodes, initially empty.
  2. Assign a tentative distance value to every node: set it to zero for the initial node and to infinity for all other nodes. Set the initial node as the current node.
  3. For the current node, calculate the tentative distance to each of its unvisited neighbors.
  4. Select the unvisited node with the smallest tentative distance, and set it as the new current node. Mark the current node as visited.
  5. Repeat steps 3 and 4 until all nodes have been visited or the target node has been visited.
  6. Construct the shortest path by tracing the path from the target node to the initial node in reverse order.
